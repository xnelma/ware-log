Tutorial for the WareLog App

For this project I want to try out the Felgo SDK. I created the initial project files using the setup wizard for a Felgo App; for a single-page application and for now without selecting plugins.

As the SDK version (4.3.1) available in the Felgo Installer at the time I am starting the project is based on Qt 6.8.3, the minimum Android version that can be built for is Android 9. The minimum target Android version is expected to be Android 12, so the Android SDK version 31 should be installed, too. This will also be the version I am building for. If the app should be published to the Play Store, note the current minimum target requirements by Google Play, which would (at the time of writing this tutorial) be Android 15, so Android SDK version 35.

Clear any component inside the AppPage component, so when you build and deploy to a device now you should see an empty window.

As a baseline the app should show a list of items with generic data fields, and allow adding and removing items. This concept can be extended to be very detailed, so in this tutorial I will show a base implementation.
