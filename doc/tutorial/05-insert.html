Insert Item

It is now possible to view the database items, and delete items. Inserting items is not implemented yet. For that we will add a floating button and the navigation to another page, write a layout for the page and implement inserting the item into the database.

Add floating button for navigation

In Felgo there is the FloatingActionButton component, which will not require manual positioning but add the button to the bottom right corner by default. Add this button to AppPage in Main.qml.

FloatingActionButton {
    iconType: IconType.plussquareo
    onClicked: console.log("clicked");
    visible: true
}

The property visible: true is necessary, as it is not the default on all platforms.

Navigate to new page

Add another qml file in the qml folder with the title EditItemView. This will be the view we will navigate to for editing an item; in this case a editing a new item to insert.

The base item should be an AppPage, and have a title to show in the toolbar. For now, just put a text item (and imports for QtQuick, QtQuick.Layouts und Felgo) in it.

AppPage {
    title: qsTr("Add New Item")

    AppText {
        text: "navigated to EditItemView"
        anchors.centerIn: parent
    }
}

In Main.qml, this page is added inside a Component. The id of the component will be used to address it when pushing it onto the stack.

Component {
    id: editItemComponent

    EditItemView {
    }
}

The stack is already available as NavigationStack by Felgo, as it was generated by the setup wizard. It can be addressed with root.navigationStack (root was the id I gave to the AppPage component). So we can now navigate to the editing page on clicking the floating button.

FloatingActionButton {
    ...
    onClicked: root.navigationStack.push(editItemComponent)
    ...
}

The NavigationStack will already handle navigation button clicks and gestures, so navigation should now fully work.
