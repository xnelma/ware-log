Insert Item

It is now possible to view the database items, and delete items. Inserting items is not implemented yet. For that we will add a floating button and the navigation to another page, write a layout for the page and implement inserting the item into the database.

Add floating button for navigation

In Felgo there is the FloatingActionButton component, which will not require manual positioning but add the button to the bottom right corner by default. Add this button to AppPage in Main.qml.

FloatingActionButton {
    iconType: IconType.plussquareo
    onClicked: console.log("clicked");
    visible: true
}

The property visible: true is necessary, as it is not the default on all platforms.
